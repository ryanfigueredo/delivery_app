# Assets

Para criar os assets necessários, você pode:

## Opção 1: Usar imagens temporárias simples

Crie arquivos PNG com as seguintes dimensões:
- `icon.png` - 1024x1024px (ícone do app)
- `splash.png` - 1242x2436px (tela de splash)
- `adaptive-icon.png` - 1024x1024px (ícone adaptativo Android)
- `favicon.png` - 48x48px (favicon web)

## Opção 2: Usar gerador online

1. Acesse: https://www.appicon.co/ ou https://www.favicon-generator.org/
2. Faça upload de uma imagem
3. Baixe os assets gerados
4. Coloque nesta pasta

## Opção 3: Criar manualmente

Use qualquer editor de imagens (Photoshop, GIMP, etc) para criar:
- Ícone: 1024x1024px com fundo transparente ou colorido
- Splash: 1242x2436px com cor de fundo #4CAF50

## Opção 4: Usar placeholder temporário

Por enquanto, o app funcionará sem os assets, mas você verá avisos.
Para produção, é necessário ter os assets corretos.
